# MedTrack

MedTrack is a web-based healthcare application that helps chronic disease patients track their medications and allows doctors to monitor whether patients are following their prescriptions.

## Problem

Many patients with chronic diseases forget to take medicines regularly, which leads to poor treatment results.

## Solution

MedTrack provides a simple platform where patients can track medications and doctors can monitor adherence.

## Features

* Medication tracking
* Reminder system
* Doctor monitoring dashboard
* Simple user interface

## Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: Firebase
Version Control: GitHub

## Future Improvements

* AI-based reminders
* Mobile application
* Health analytics
